# Greenhouse Simulator 🌿
Интерактивный симулятор теплицы на Java.

## Описание
Проект представляет собой консольное приложение для управления ростом растений в реальном времени с учетом погодных условий.

## Технологии
- **Язык:** Java (JDK 24)
- **База данных:** SQLite (для сохранения прогресса)
- **Библиотеки:** JDBC
- **Особенности:** Использование многопоточности (Threads) для симуляции погоды и роста.
